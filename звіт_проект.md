## 📝 СТИСЛИЙ ВИКЛАД КОМАНДНОГО РЕЗУЛЬТАТУ
В межах командного проекту було розроблено клієнт-серверний застосунок, який складається з високопродуктивного бекенду на мові **C++** та преміального інтерактивного фронтенду (веб-інтерфейсу) на базі **HTML5, CSS3 та JavaScript (Glassmorphism)**.

### Основні функціональні можливості застосунку:
1. **Багатопотоковий HTTP-сервер на C++ (Winsock2)**:
   - Обробляє клієнтські запити асинхронно у відокремлених потоках (`std::thread`).
   - Реалізує автоматичний пошук вільного порту в діапазоні `8080-8100` для запобігання конфліктам портів.
   - Забезпечує стабільність зв'язку при speculative-з'єднаннях сучасних браузерів за рахунок індивідуальнихreceive-таймаутів.
2. **Модуль обробки BMP-зображень**:
   - Робота з 24-розрядним колірним форматом BMP без стиснення.
   - Читання, парсинг заголовків (`BITMAPFILEHEADER`, `BITMAPINFOHEADER`), модифікація піксельних масивів з урахуванням вирівнювання рядків (BMP Padding) та перезапис файлу на диск.
3. **Генератор математичних бітових карт (3 алгоритми та 3 колірні схеми)**:
   - *Інтерференція хвиль (Wave Interference)* з колірними схемами: Неоновий кіберпанк, Золотий ембер, Глибоке море.
   - *Математична плазма (Plasma)* з колірними схемами: Полум'я, Аврора, Психоделіка.
   - *Побітові візерунки (Bitwise/XOR)* з колірними схемами: Матриця, Монохром, Фіолетовий неон.
4. **Стеганографія найменшого значущого біта (LSB)**:
   - Вкладення секретного текстового повідомлення у молодші біти колірних каналів (B, G, R) BMP-карти.
   - Додавання нуль-термінатора (`\0`) для точного визначення кінця повідомлення під час зчитування (декодування).
5. **Авторизація та збереження історії сесії (JSON DB)**:
   - Реєстрація та вхід користувачів із використанням хешування паролів.
   - Збереження у файлі `users.json` та автоматичне відновлення останніх налаштувань користувача (останній відкритий файл, обраний алгоритм генерації та колірна схема).
   - Відображення історії останніх 3-х дій (файли, режими, вкладені тексти, зчитані тексти) у вигляді клікабельних списків.

## 🛠️ РОЗГОРНУТИЙ ВИКЛАД ОСОБИСТОГО ВНЕСКУ УЧАСНИКА

Я виконував роль **Бекенд-розробника (Backend C++ Developer / Integration Engineer)** та відповідав за архітектуру проекту, алгоритмічну частину, збереження даних та стабільність клієнт-серверної взаємодії.

### Мої особисті завдання та досягнення в проекті:
1. **Проектування та розробка веб-сервера C++**:
   - Написав низькорівневу логіку сокетів Winsock2 для Windows (`listen`, `accept`, `recv`, `send`).
   - Перевів архітектуру сервера на багатопотокову основу (`std::thread`, `std::mutex`), що дозволило запобігти зависанню сервера при одночасному завантаженні сторінки та зображень.
   - Додав захисний механізм автовизначення неінтерактивного режиму роботи (запуск під IDE) та реалізував API-ендпоінт для прямого завантаження (`/api/upload-file`) та скачування файлів у браузер, що повністю захистило додаток від системних блокувань Windows API (`GetOpenFileName` / `GetSaveFileName`).
2. **Реалізація математичної генерації візерунків**:
   - Написав функції розрахунку інтенсивностей для інтерференції хвиль, плазми та побітових візерунків.
   - Налаштував перетворення інтенсивності у кольорові канали RGB відповідно до обраної користувачем палітри.
3. **Розробка стеганографічного модуля LSB**:
   - Реалізував побітове вкладення символів у молодші біти пікселів BMP.
   - Забезпечив коректне зчитування прихованого тексту з виявленням нуль-термінатора.
4. **Створення бази даних та системи сесій**:
   - Написав легковажний JSON-серіалізатор та десеріалізатор для збереження профілів користувачів (`users.json`).
   - Написав алгоритми безпечного збереження паролів (хешування) та ведення історії останніх 3-х дій для кожного профілю.
5. **Інтеграція та тестування**:
   - Розробив вбудований автоматизований тест-сьют (запуск через `.\app.exe --test`), який тестує основні модулі бекенду: BMP-парсер, LSB-стеганографію, генерацію плазми та роботу з JSON.
   - Здійснив фінальну збірку проекту за допомогою MinGW `g++` та налаштував Git/GitHub репозиторій для спільної роботи.
